# 🤖 Лабораторна робота №3 
**Курс:** «Моделі генеративного ШІ»

Цей репозиторій містить матеріали та template звіту для виконання третьої лабораторної роботи з курсу, присвяченого моделям генеративного штучного інтелекту.

## 📄 Опис завдання

Детальні умови, мета роботи, теоретичні відомості та покрокові інструкції знаходяться у прикріпленому документі:
👉 [**Лабораторна_робота_3.pdf**](./Лабораторна_робота_3.pdf)

*Перед початком роботи обов'язково ознайомтеся з цим файлом.*

## 📊 Фіксація результатів

Усі результати виконання лабораторної роботи повинні бути задокументовані у файлі [**OUTPUT.md**](./OUTPUT.md). У цьому файлі вже підготовлено базовий темплейт (шаблон) для заповнення.

**Вимоги до оформлення звіту:**
* **Текстовий опис:** Кожен етап роботи має супроводжуватися стислим, але зрозумілим описом ваших дій (наприклад, які промпти використовувались, які параметри моделей змінювались).
* **Скріншоти:** Обов'язково додавайте візуальні підтвердження роботи (скріншоти інтерфейсів). 
  > *Порада:* Рекомендується створити окрему папку `images/` або `assets/` у цьому репозиторії для зручного зберігання всіх скріншотів, щоб не засмічувати кореневу директорію.

## 🚀 Порядок виконання (Workflow)

1. Уважно прочитайте завдання у файлі `Лабораторна_робота_3.pdf`.
2. Виконайте необхідні практичні кроки у відповідних середовищах або з використанням API генеративних моделей.
3. Робіть скріншоти ключових етапів та результатів генерації.
4. Відкрийте `OUTPUT.md` та заповніть його згідно з існуючим темплейтом, описуючи процес та вставляючи збережені скріншоти.
5. Збережіть зміни, зробіть `commit` та `push` у свій віддалений репозиторій.
